## Project Overview

This project is a React podcast browsing application that allows users to explore podcasts using search, sorting, genre filtering and pagination.

The interface updates dynamically while users interact with the controls.

## Features

### Search

Users can search podcasts by typing any part of the podcast title.

### Sorting

Podcasts can be sorted by:

* Newest
* Title A–Z
* Title Z–A

### Filtering

Users can filter podcasts by genre.

### Pagination

Podcasts are displayed across multiple pages to improve readability and performance.

### Responsive Layout

The podcast grid adapts to desktop, tablet and mobile screens.
