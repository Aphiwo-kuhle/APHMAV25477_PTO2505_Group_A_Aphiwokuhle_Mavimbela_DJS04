Project Overview

This project is a React podcast browsing application that allows users to explore podcasts through searching, sorting, filtering and pagination.

The application fetches podcast preview data from the podcast API and dynamically updates the interface as users interact with the controls.

Features
Search

Users can search podcasts by typing any part of the podcast title. Results update dynamically while typing.

Sorting

Podcasts can be sorted by:

Newest updated

Title A–Z

Title Z–A

Filtering

Users can filter podcasts by genre.

Pagination

Podcasts are displayed in smaller groups across pages for better performance and readability.

State Synchronization

Search, sorting, filtering and pagination work together without resetting each other.
